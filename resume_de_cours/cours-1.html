Cours n°1: les variables

HTML

HTML est un langage de description balisé. Il permet de décrire le contenu d'une page web, sans se soucier de la mise en forme.

Ecrire le code HTML

Le code HTML s'écrit dans un fichier d'extension .html. On appelle souvent le fichier principal index.html.

Balises et attributs HTML

On utilise des balises pour décrire le contenu de la page web. Cahque balise possède un nom et représente un certain élément (titre, paragraphe, image, ...). Elles possède un sens (sémantique) et souvent une mise en page par défaut.

Il ne faut pas utiliser une balise pour changer l'apparence du texte. C'est la fonction du langage CSS que nous verrons plus tard. On utilise une balise pour insérer un élément particulier.

Certaines balises peuvent contenir d'autres éléments (balises ou texte). Dans ce cas, elles commencent par une balise ouvrante et se terminent par une balise fermante.


<nomDeLaBalise> <!--Balise ouvrante-->
    Contenu
</nomDeLaBalise> <!--Balise fermante-->

Une balise ne peut être fermée que si les balises qu'elle contient sont fermées également.

Le texte entre <!-- et --> est ce qu'on appelle un commentaire. Il s'agit d'une indication laissée par le programmeur pour les personnes lisant le programme. Ces informations ne seront pas lues par le naviateur. Attention, puisque tout le monde à accès à ce texte, il ne faut pas y mettre d'informations sensibles (comme des mots de passe).
