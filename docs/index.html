Bienvenue aux élèves d'Acapelart !

Sur ce site, vous trouverez toutes les ressources pour continuer de progresser:

  • Les différents projets qui ont été réalisés avec leur code source.
  • Les résumés de tous les cours.
  • Un guide pour bien débuter la programmation web.
  • Un référence pour chaque langage, contenant les différents éléments vus. Une sorte en mémo.
  • Les objectifs de la prochaine séances ainsi que les ressources à télécharger.

J'espère que vous trouverez ces ressources utiles. Gardez en tête que la clef pour progresser est la régularité. Je compte sur vous et je vous souhaite une bonne programmation!

Pour la prochaine fois.

Une semaine entre deux activités ça peut être long... et parfois on oubli tout! Voici quelques conseils pour attendre samedi prochain:

  • Lire les notes du cours qui sont sur ce site. Comme ça, on oublie rien!
  • Tester les projets du dernier cours, lire le code et essayer de le comprendre, même si ce n'est pas simple.
  • Et pourquoi ne pas essayer de réaliser son propre projet? Attention à ne pas entreprendre quelque chose de trop ambitieux!
