Comment jouer?

  1. Entrer un nombre au clavier. Si le caractère entré n'est pas un nombre, ou si le nombre est trop grand, vous serez prévenu. Vous pouvez utiliser la touche effacer pour modifier votre nombre.
  2. Quand votre nombre est prêt, appuyer sur entrée.
  3. L'ordinateur va vous dire si le nombre qu'il a choisi est supérieur, inférieur ou si vous avez gagné.
  4. Pressez entrée pour continuer de jouer. Si vous n'avez pas gagné, le nombre reste le même, si vous avez gagné, le nombre change.
